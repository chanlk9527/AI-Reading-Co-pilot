# Startup Scripts

Scripts are separated by OS to keep the root directory clean:

- `scripts/windows/*.bat`: Windows startup scripts
- `scripts/mac/*.sh`: macOS/Linux startup scripts

Common commands:

- Start backend: `scripts/windows/start_server.bat` or `./scripts/mac/start_server.sh`
- Start frontend: `scripts/windows/start_frontend.bat` or `./scripts/mac/start_frontend.sh`
- Restart all: `scripts/windows/restart_all.bat` or `./scripts/mac/restart_all.sh`
- Stop all: `scripts/windows/stop_all.bat` or `./scripts/mac/stop_all.sh`
